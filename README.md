\# 🔐 Générateur de mot de passe



Petit outil en ligne de commande pour générer des mots de passe sécurisés.



## Fonctionnalités
- Longueur personnalisable (8 à 32 caractères)
- Option chiffres et/ou caractères spéciaux
- Génération cryptographiquement sûre (module `secrets`)


\## Utilisation

```bash

python generateur\_mot\_de\_passe.py

```



\## Prérequis

Python 3.6+
